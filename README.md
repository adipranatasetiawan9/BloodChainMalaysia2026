# 🩸 BloodChain Malaysia 2026

## Digital Supply Chain Transfusi Darah Berbasis Blockchain

BloodChain Malaysia 2026 merupakan aplikasi simulasi Digital Supply Chain Transfusi Darah berbasis Blockchain yang dikembangkan sebagai proyek mata kuliah Anti TI Semester 6.

## Fitur Aplikasi

- Login Administrator
- Dashboard Modern
- CRUD Data Pendonor
- CRUD Data Rumah Sakit
- CRUD Stok Darah
- CRUD Permintaan Darah
- Blockchain Simulation
- Supply Chain Transfusi Darah
- Riwayat Distribusi
- Scan QR Code (Simulasi)
- Local Storage

## Teknologi

- HTML5
- CSS3
- JavaScript
- Local Storage
- Blockchain Simulation

## Cara Menjalankan

1. Install Python di Termux.
2. Masuk ke folder proyek.

```bash
cd ~/BloodChainMalaysia2026
```

3. Jalankan server.

```bash
python -m http.server 8080
```

4. Buka browser.

```
http://127.0.0.1:8080
```
